## Selected Problem
Rapid Crisis Response: Accelerated Emergency Response and Crisis Coordination in Hospitality.

Hospitality venues can face emergencies where information is fragmented between guests, staff and responders. The challenge asks teams to design a system that detects, reports and synchronizes response efforts across this decentralized ecosystem.

## Product Hypothesis
If guests can report emergencies quickly and staff receive Gemini-structured incident summaries, severity, location, next actions and live status, then hospitality venues can reduce confusion and coordinate faster during high-stress events.

## Target Users
| User | Need |
| --- | --- |
| Guest | Fast SOS/reporting without installing an app. |
| Front desk / operations manager | Single dashboard for all active incidents and team assignments. |
| Floor staff / security | Clear tasks, location and next action guidance. |
| First responder liaison | Structured report with incident type, severity, location and current status. |

## Differentiation
- QR-first guest intake works without requiring app installation.
- Gemini converts unstructured, multilingual reports into structured response plans.
- Staff dashboard keeps incident status, map, tasks and communication in one place.
- MVP is lightweight enough for small hotels but architected for Cloud Run/Firebase scaling.
